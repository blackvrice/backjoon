# BOJ LSP Workspace

This folder contains basic Language Server workspace settings for:

- C++17: clangd (`.clangd`)
- Java 17: Eclipse JDT LS (`.project`, `.classpath`, `.settings`)
- C#: Roslyn / OmniSharp (`Directory.Build.props`, `.omnisharp/omnisharp.json`, `csharp/BoJWorkspace.csproj`)
- Dart: Dart Analysis Server (`pubspec.yaml`, `analysis_options.yaml`)
- Python 3.12: Pyright (`pyrightconfig.json`)
- JavaScript: TypeScript Language Server (`jsconfig.json`, `package.json`)

Place this `.lsp-workspace` directory at your project root.
